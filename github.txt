# Aprendendo Git e Github
Vamos iniciar dizendo que usaremos apenas o terminal de comandos
vamos iniciar instalando o git.
@>> sudo apt install git
Para ver se instalou como deveria.
@>> git --version | retorna a versão do git
crie a pasta do seu projeto e digite:
@>> git init | inicializa o projeto
@>> git add nome_do_arquivo | adiciona o arquivo ao stage
@>> git rm nome_do_arquivo | retira o arquivo do stage

@>> git add . | adicona todos os arquivos ao git necessário antes do commit(sempre)
@>> git commit -m "o que foi feito" | sobe par ou altera seu projeto para o git
@>> git push | sincroniza as versôes fazer sempre após o commit

Da para fazer o seguinte:
@ git add . && git commit -m "mensagem" && git push | parece uma boa idéia, sera? kkk

@>> git staus | retorna o status do seu projeto

MUITA ATENÇÃO A PARTIR DAQUI
@>> git log --oneline | retorna a lista de commits e seus ids
@>> git checkout id_commit| leva você ao commit solicitado observe que os arquivos estão como naquele commit cuidade para não de confundir.
Para retornar ao commit atual digite:
@>> git checkout master ou main


@>> git reverse id_commit | usado para reverter a última alteração feita no arquivo
@>> git reset id_commit | usado para apagar todos os commits a partir daqule commit
@>> git reflog lista todos os commits feitos

@>> git branch nome_da_branch | cria uma nova branch
como estamos no terminal precisamos publicar branch digite

@>> git push -u origin nome-da-branch | Publica a branch
@>> git branch | retorna as branchs existentes no repositório

@>> git checkout nome_da_branch | entra na brunch desejada

A partir deste ponto todas os commits serão realizados nesta branch
Imagine um projeto de ERP, podemos ter uma branch clientes uma fornecedores, contas a pagar e por ai
em diante e cada desenvolvedor trbalhando em uma delas e testanto seu trabalho independente
dos outros.
A essa altura fica claro que em algum momento precisaremos juntar as coisas ai é que surge o merge

Primeiro, se já não estiver vá para a branch master ou main

digite:

@>> git merge teste
